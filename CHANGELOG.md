## [1.0.0] - 2026-01-27

### Added
- Initial release of Enhanced Schema Plugin documentation website
- Home page with hero section and feature overview
- Features page with detailed plugin capabilities
- Installation page with setup instructions
- Comparison page vs SchemaWriter.ai
- Dark theme with gradient design
- Responsive navigation
- Download buttons for plugin versions
- Hash-based routing for static hosting compatibility
- Pre-built production files in `dist/` and `docs/` folders
- Complete documentation (README, INSTALLATION, DEPLOYMENT, CONTRIBUTING)
- GitHub Pages deployment
- Feature comparison markdown document

### Technical Stack
- React 19 with TypeScript
- Tailwind CSS 4 for styling
- Vite for build tooling
- shadcn/ui component library
- Wouter for client-side routing
- Hash-based routing for file:// protocol support

## Version History

### Version Naming Convention

- **Major (X.0.0):** Breaking changes, major redesigns
- **Minor (1.X.0):** New features, non-breaking improvements
- **Patch (1.0.X):** Bug fixes, minor updates

### Release Process

1. Update CHANGELOG.md
2. Update version in package.json
3. Create git tag: `git tag -a v1.0.0 -m "Release v1.0.0"`
4. Push tag: `git push origin v1.0.0`
5. Build: `npm run build`
6. Deploy to production
