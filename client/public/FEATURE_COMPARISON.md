## Detailed Feature Comparison

### Webpage Schema Generation

**SchemaWriter.ai:**
- Generates schema for up to 3 keywords at a time
- 2-minute report generation
- Requires manual copying and pasting into website
- Limited to subscription plan runs per month

**Enhanced Schema Plugin:**
- Automatic generation for all pages and posts
- Instant generation on page save
- Automatic insertion into page head
- No limits on number of pages

### Entity Detection

**SchemaWriter.ai:**
- Analyzes up to 10 competitors
- Uses Google Search API for competitor identification
- Entity filter with topical correlation
- AUTO mode mixes own and Google-preferred entities

**Enhanced Schema Plugin:**
- Analyzes page content directly
- Extracts from headings, bold text, and links
- Scoring algorithm for relevance
- Manual entity addition and removal
- Cached entities for performance

### Georadius Generation

**SchemaWriter.ai:**
- Available only in Agency plan ($119/month)
- Less than 30 seconds generation time
- Interactive editor for cities/postal codes
- Includes Wikipedia, Wikidata, Google Maps URLs

**Enhanced Schema Plugin:**
- Available in all installations (no extra cost)
- Automatic generation with Google Places API
- Visual interface for managing locations
- Same URL enrichment as SchemaWriter.ai
- Cached results for performance

### Local Business Schema

**SchemaWriter.ai:**
- Complete local business markup
- Google Places API integration
- Georadius included in Agency plan
- Manual schema copying required

**Enhanced Schema Plugin:**
- Complete local business markup
- Google Places API integration
- Georadius included by default
- Automatic output on all pages
- Per-location support for multi-location businesses

## Limitations vs SchemaWriter.ai

### 1. Schema Iframe Feature
- SchemaWriter.ai offers schema iframe generation for backlinks
- Not yet implemented in plugin
- Can be added as future enhancement

### 2. Competitor Analysis
- SchemaWriter.ai analyzes competitor schemas
- Plugin focuses on own content
- Could be added in future version

### 3. Multi-language Support
- SchemaWriter.ai supports 17 languages
- Plugin currently English-only
- Can be extended for other languages

### 4. Hosted Service
- SchemaWriter.ai is fully hosted (no installation)
- Plugin requires WordPress installation
- Trade-off for control and cost savings
